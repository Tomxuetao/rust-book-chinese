静态页面构建与文档撰写 #

构建 #

我们采用 vitepress 打包出静态网页。在这之前，你需要安装 Nodejs。

cd 到项目目录，安装包

bash
pnpm install

构建好的静态文档会出现在 "./.vitepress/dist" 中

bash
pnpm run build

文档撰写 #

vitepress 会启动一个本地服务器，并在浏览器对你保存的文档进行实时热更新。

bash
pnpm run dev
